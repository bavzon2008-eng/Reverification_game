# Reverification Game 🎮

A simple browser-based verification game developed using HTML, CSS, and JavaScript.

## Features
- Interactive gameplay
- Responsive user interface
- Fast and lightweight
- Pure HTML, CSS, and JavaScript (No frameworks)

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Clone the repository
2. Open `index.html` in any web browser

## Project Structure
```
Reverification_game/
│── index.html
│── style.css
│── script.js
```
